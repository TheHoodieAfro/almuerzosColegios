# Requerimientos

- La aplicacion es para llevar un control del PAE, programa de alimentación
escolar, hay que meter una base de datos de los estudiantes desde 6 hasta 11
- Debe ser capaz de generar reportes de asistencia diaria, semanal y mensual
- La idea es que el estudiantes escriba un codigo y de una quede registrado con
dia y hora
- Toda la informacion del estudiande

## Tech stack

### Front-end

Vite
Vue + Tailwind CSS

### Back-end

Python + FastAPI
API testing with Yaak

### DB

SQLAlchemy as ORM
SQLite with DB Browser for SQLite (<https://sqlitebrowser.org/>)

### Deployment

Docker compose file
Create installation script or intruction manual

### Enviroment

Preferably linux, be it x86 or ARM (raspberry PI)
